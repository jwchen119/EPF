<objective>
Add three gray shade options (Dark Gray, Gray, Light Gray) to the overlay color system so the
date/location overlay's background, text, and border colors can use grays in addition to the
existing 6-color T133A01 palette. Grays are overlay-only RGB values drawn before quantization;
on the e-paper they nearest-neighbor to black or white (D-02), which is expected.

Purpose: Give the user finer visual control over overlay contrast without touching the hardware
quantization palette.
Output: 3 new entries in OVERLAY_COLORS, an updated contract test, and 3 new option entries in
each of the three color dropdowns in the settings UI.
</objective>
