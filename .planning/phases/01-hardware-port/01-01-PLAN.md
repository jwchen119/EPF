- epd7in3e.h: T133A01 resolution + color code constants (Epd class removed)

</objective>

   - DC_PIN    = 10    (GPIO10)
   - CS_PIN    = 44    (GPIO44 / D7 on XIAO header)
   - CS1_PIN   = 41    (secondary chip-select for T133A01 dual-driver IC)

3. Include the Seeed_GFX and TFT_eSPI headers so downstream files that include epdif.h
   automatically get the display library:
   - #include <Seeed_GFX.h>
   - #include <TFT_eSPI.h>

4. Keep the standard header guard (#ifndef EPDIF_H / #define EPDIF_H / #endif).

5. Add a comment block at the top documenting the target hardware:
   "EE02 HAT + XIAO ESP32-S3 Plus + Seeed 13.3 inch Color Eink (T133A01)"
   and the source reference: "Seeed_GFX EPaper_Board_Pins_Setups.h USE_XIAO_EPAPER_DISPLAY_BOARD_EE02"

Write the complete new file.
