--- CHANGE 2: Replace depalette_image() index-shift line ---
In depalette_image() (line ~166), remove:
  `indices[indices > 3] += 1  # Simulate the code from the C`

This line was WaveShare-specific — it skipped the CLEAN color at index 3.
The T133A01 has no CLEAN/skip code. Removing this line makes indices a plain
0–5 range directly into the palette.

The function after the change:
```python
def depalette_image(pixels, palette):
    palette_array = np.array(palette)
    diffs = np.sqrt(np.sum((pixels[:, :, None, :] - palette_array[None, None, :, :]) ** 2, axis=3))
    indices = np.argmin(diffs, axis=2)
    return indices
```

--- CHANGE 3: Replace convert_to_c_code_in_memory() nibble packing ---

**IMPORTANT — nibble_map ordering:** The RESEARCH.md Open Questions item 4 suggests a lookup
`[0x0, 0x2, 0x6, 0xB, 0xD, 0xF]` written for T133A01-native (white-first) index order.
Do NOT use that suggestion. The palette in Change 1 is defined with black at index 0.
The nibble_map `[0xF, 0x0, 0xB, 0x6, 0xD, 0x2]` is correct for that black-first palette
order and must be used exactly as written.

The current packing uses raw palette indices as nibble values:
  `(indices[y, x] << 4) | indices[y, x + 1]`
This is wrong for T133A01 because palette indices 0–5 do NOT match the T133A01
nibble codes (0xF, 0x0, 0xB, 0x6, 0xD, 0x2).

Replace the bytes_array list comprehension in convert_to_c_code_in_memory() with
an explicit nibble lookup:

```python
def convert_to_c_code_in_memory(image_data):
    """ Convert image to C code in memory — T133A01 nibble encoding """
    pixels = np.array(image_data)

    # Nearest-neighbor palette quantization
    indices = depalette_image(pixels, palette)

    # T133A01 nibble codes indexed by palette position:
    # palette[0]=black→0xF, [1]=white→0x0, [2]=yellow→0xB,
    # [3]=red→0x6, [4]=blue→0xD, [5]=green→0x2
    nibble_map = [0xF, 0x0, 0xB, 0x6, 0xD, 0x2]

    height, width = indices.shape
    bytes_array = [
        (nibble_map[indices[y, x]] << 4) | nibble_map[indices[y, x + 1]]
        if x + 1 < width
        else (nibble_map[indices[y, x]] << 4)
        for y in range(height)
        for x in range(0, width, 2)
    ]

    # Generate hex CSV output
    output = io.StringIO()
    for i, byte_value in enumerate(bytes_array):
        output.write(f"{byte_value:02X},")
        if (i + 1) % 16 == 0:
            output.write("\n")
    output.write("};\n")

    result = output.getvalue().encode('utf-8')
    output_bytes = io.BytesIO(result)
    output_bytes.seek(0)
    return output_bytes
```
