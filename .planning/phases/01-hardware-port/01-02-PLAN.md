<objective>
Rewrite epd7in3e.ino to use Seeed_GFX (TFT_eSPI) instead of the WaveShare Epd driver.
Update config.h to remove the oversized BUFFER_SIZE. Remove the battery voltage guard.

application logic. The image download path must switch from streaming byte-by-byte to the
display (epd.SendData) to buffering the full 960 KB frame in PSRAM and pushing it at once
via epaper.pushImage(). The deprecated sleep API and battery guard are also fixed here.

Output:
- epd7in3e.ino: full rewrite of display-related code, battery guard removed, sleep API updated
- config.h: BUFFER_SIZE updated to HTTP chunk size (16384), WAKEUP_PIN kept at GPIO_NUM_2
